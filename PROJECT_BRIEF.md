# PROJECT BRIEF: 个人高端简历官网
## 项目目标
开发一个 **视觉炫酷、高级质感、符合 2025 现代审美** 的个人技术简历/作品集单页网站，纯前端、可直接部署、响应式、动画流畅。

## 核心页面模块
1. Hero 区域（全屏动态背景 + 姓名 + 职业定位 + 快速入口）
2. About Me（个人介绍、技术方向、社交链接）
3. Resume（教育经历、工作经历、技能体系、证书）
4. Projects（项目卡片、技术栈标签、演示链接、GitHub 链接）
5. Contact（极简联系区域）

## 强制视觉风格（高级炫酷）
- 风格：现代极简 + 玻璃拟态（Glassmorphism）+ 渐变色
- 动态：粒子背景 / 流体背景 二选一
- 动画：滚动入场动画、3D 卡片悬停、平滑过渡
- 主题：支持暗黑模式 / 明亮模式自动切换
- 设计级别：Awwwards 级别高端前端

## 必须实现的功能
1. 完美响应式（手机 / 平板 / PC 全适配）
2. 平滑滚动、固定导航栏
3. 项目筛选功能（按技术栈分类）
4. 简历 PDF 一键下载按钮
5. 鼠标轻量跟随交互
6. 技能条动态加载动画
7. 页面加载动画
8. SEO 友好

## 技术栈（严格遵守）
- 框架：React 18
- 构建：Vite
- 样式：Tailwind CSS
- 动画：Framer Motion
- 无后端、纯静态站点

## 代码要求
- 生产级别、结构清晰、模块化
- 注释完整、可维护、无冗余
- 可直接部署到 GitHub Pages / Vercel

## 开发规则（Codex 必须遵守）
1. 先搭建整体架构，再开发页面，最后优化动画
2. 自动分阶段开发，无需人工干预
3. 遇到错误自动重试
4. 通宵执行，完成全部功能
5. 保证 UI 高级、炫酷、无 bug
